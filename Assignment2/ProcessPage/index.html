Accessability Process Page

Use Cases

  • Finding out what bands are playing
  • Finding out the dates of the event
  • Finding out the location of the event

Process for finding the location:

  • View headings
  • Find heading for Directions
  • Go forward one, gives a text description of the place.
  • However, focusing on the google map below it create a garbled mess.
  • Can be problematic since it's at the very bottom

Solutions

  • Added a line at the top description which says the time and date right away, so users don't have to go all the way to the bottom.
  • Added a title to the google maps iframe which the screenreader reads, telling the user the address again
